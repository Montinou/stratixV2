## Testing Methodologies

### Moderated Testing
```markdown
Setup:
- Define tasks
- Recruit participants
- Prepare materials
- Conduct sessions
- Analyze findings

Metrics:
- Task completion rate
- Time on task
- Error frequency
- Satisfaction score
- Qualitative feedback
```

### Unmoderated Testing
```markdown
Tools: UserTesting, Maze
Benefits:
- Larger samples
- Natural environment
- Cost effective
- Quick turnaround

Limitations:
- Less insight depth
- No follow-up questions
- Technical issues
- Participant quality
```

### A/B Testing
```markdown
Process:
1. Identify variable
2. Create variants
3. Split traffic
4. Measure results
5. Statistical analysis
6. Implement winner
```
